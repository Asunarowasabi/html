uec_rescue

  • よく使うUnixコマンド
    1. cd:ディレクトリ移動用。後ろに行きたい場所を書く。
    2. ls:あるディレクトリ内のファイル等の一覧を表示する。階層構造を記述すれば別ディレクトリでもOK。
    3. pwd:今いるディレクトリ(カレントディレクトリ)を表示する。どこだか分からなくなったときに。
    4. chmod:ファイルの各種権限を変更する。「chmod 777」にするといちいち権限で悩む必要がないからいいぞ(やめなさい)
    5. mv:ファイル移動用。「mv 移動したいファイル名 移動させたい場所」のように使う。後ろ忘れやすいので注意！ファイル名変えたいときもこのコマンド。
    6. cp:ファイル複製用。mvと同じ要領。
    7. rm:ファイル消去用。ディレクトリはきちんと整理しましょう。ディレクトリを消すなら「-r」オプションで。
    8. platex:texファイルをdviファイルに整形するときに使う。エラーはちゃんと読もう。Googleで調べればだいたい対処できる。
    9. pxdvi:dviファイル閲覧に使用。
    10. dvipdfmx:dviファイルをpdfファイルに変換するときに使う。「dvipdf」だと実行はできるが文字化けするので注意！
    11. evince:pdf閲覧用コマンド。
    12. echo X | mail -s X -a ファイル名 自分のメルアド:ファイルを手元に持ってくるためのコマンド。とにかく覚えにくい。SFTP環境構築構築しような
    13. emacs:いつもの
    14. gimp:読んで字のごとく。Paint.netは神
    15. gv:epsファイルを見る時の命令。
    16. exit:ログアウト用。
